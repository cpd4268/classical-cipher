# Caesar-cipher
## Overview
This repository has Caesar-sipher to be programmed with Python3.

## Description
- keygen.py:
    - output:  
    key (n  <- {1, 2, ... 25})

- Enc.py:
    - input:  
    plain text  
           key  
    - output:  
    cipher text  

- Dec.py:
    - input:  
    cipher text  
           key  
    - output:  
    plain text

## Requirement
Python version (>=3.5)
